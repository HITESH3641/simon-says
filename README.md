# 🎮 Simon Says Game

A browser-based implementation of the classic **Simon Says** memory game built using **HTML**, **CSS**, and **JavaScript**. Test your memory by repeating an increasingly long sequence of colors while trying to achieve the highest score.

---

## 🚀 Features

* 🎲 Random color sequence generation
* 🧠 Memory-based gameplay
* 📈 Progressive difficulty (new color every level)
* 🏆 High score tracking
* ✨ Button flash animations
* 📱 Desktop and mobile support
* 🎨 Simple and responsive interface
* ⚡ Instant game restart after game over

---

## 🎮 How to Play

1. Press any key (or tap the screen on mobile) to start the game.
2. Watch the sequence of flashing colors.
3. Repeat the sequence by clicking the colored buttons.
4. Every new level adds one additional color.
5. Clicking the wrong color ends the game.
6. Try to beat your highest score!
